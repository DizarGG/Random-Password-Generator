---

## Описание

GUI-приложение для генерации случайных паролей с настройкой параметров,
оценкой стойкости и сохранением истории.

Написано на **Python 3.10+** с использованием стандартной библиотеки:
`tkinter`, `random`, `string`, `json`.

---

## Возможности

| Функция                  | Описание                                        |
|--------------------------|-------------------------------------------------|
| 🎚️ Ползунок длины       | От 4 до 64 символов                             |
| ☑️ Чекбоксы символов    | Прописные, строчные, цифры, спецсимволы         |
| 💪 Индикатор стойкости  | Оценка 0–100 с цветовой шкалой                  |
| 📋 История               | Таблица всех паролей, двойной клик = копировать |
| 💾 Сохранение в JSON     | Автоматически в `history.json`                  |
| 📎 Копирование           | В буфер обмена одной кнопкой                    |
